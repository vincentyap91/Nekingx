# Fonts

Place licensed **Geogrotesque** font files here to enable `@font-face` loading.

Expected filenames (woff2 preferred):

- `Geogrotesque-Regular.woff2` (weight 400)
- `Geogrotesque-SemiBold.woff2` (weight 600)
- `Geogrotesque-Bold.woff2` (weight 700)

Do not commit or use unlicensed font downloads. Until licensed files are added, the site falls back to Inter / PingFang SC / Arial.
